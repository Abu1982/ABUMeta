## 4. Social Preview Visual Spec / 社交预览图设计说明

### Canvas

- Size: `1280 x 640`
- Safe margin: `80 px`
- Export: `PNG`

### Visual Direction

- Tone: engineered, cerebral, institutional, not sci-fi neon clutter
- Mood: serious research prototype with operational clarity
- Avoid: generic purple gradients, default AI-glow clichés, random wireframe overload

### Layout

- Left 55%: headline block and support lines
- Right 45%: abstract module lattice or orbit-based system diagram
- Bottom strip: 5 capability labels separated by dots

### Typography

- English headline: `Space Grotesk Bold` or `Sora SemiBold`
- Chinese secondary line: `Noto Sans SC Medium` or `Source Han Sans Medium`
- Metadata line: mono or semi-mono style such as `IBM Plex Mono`

### Color System

- Background base: `#0F1722`
- Secondary field: `#162235`
- Accent teal: `#39C6B4`
- Accent amber: `#F3B94C`
- Soft steel: `#8EA3B7`
- White text: `#F5F7FA`

### Graphic Elements

- Use 5 connected nodes to represent the architecture pillars
- Suggested labels: `Memory`, `Psyche`, `Decision`, `Execution`, `Governance`
- Add thin orbital curves or disciplined grid lines rather than flashy particles
- Optional small bilingual caption in the lower right: `Open-Source Demo / 开源演示版`

### Composition Notes

- Keep the word `ABUMeta` visually dominant
- Do not overfill the right side; preserve negative space
- Prioritize legibility in GitHub link previews on mobile
