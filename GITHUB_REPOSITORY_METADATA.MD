# GitHub 仓库元数据建议

## 仓库名称

`ABUMeta`

## 仓库简介文案（短）

一个可演示的类人自治 Agent 开源副本，包含记忆、心理、决策、抓取与治理骨架，适合作为 AGI 工程化原型与研究型演示项目。

## 仓库简介文案（中）

ABUMeta 是从 ABU 主开发仓导出的可演示开源版，保留了记忆、心理引擎、决策引擎、抓取执行与治理结构，同时移除了数据库、缓存、日志、抓取结果和历史运行资产，适合 GitHub 展示、二次开发和工程研究。

## 仓库首页长描述建议

ABUMeta 提供一个面向“高级自治 Agent / 准 AGI 工程原型”方向的可演示开源副本。它保留了核心认知模块、最小演示链路和基础测试，同时将第三方依赖、运行资产、外部抓取结果与主开发仓私有状态做了隔离。适合希望研究记忆系统、心理引擎、规则化决策、影子执行与开源演示型自治体架构的开发者。
