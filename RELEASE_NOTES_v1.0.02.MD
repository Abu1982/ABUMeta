# Release Notes

## 标题建议

`v1.0.02 - GitHub 首发元数据与最终公开前收口`

## 发布说明正文

### 本次更新

- 补充 GitHub 首发元数据建议，新增仓库简介文案、Topics 建议和发布说明模板
- 完成最终公开前收口，继续强化 `.gitignore`，减少误提交运行资产、缓存和本地文件的风险
- 保留本地双击安装体验，同时将 Windows 快捷方式从 Git 跟踪中移除，避免把本地化文件推上公开仓库
- 维持开源演示版的最小可运行结构，并保留 smoke 测试能力

### 适合谁

- 希望直接在 GitHub 展示一个可运行自治 Agent 架构副本的人
- 希望在不接触主开发仓私有资产的前提下研究 ABU 骨架的人
- 希望在公开仓库中继续二次开发或提交 PR 的贡献者

### 验证

- 使用副本虚拟环境运行 `tests/test_smoke_demo.py`
- 确认 README、许可证、第三方说明、更新日志、Issue/PR 模板齐备
- 确认 Git 标签与版本文件同步到本次版本

### 注意

- 本仓库是开源演示版，不包含主开发仓数据库、抓取结果、缓存与历史运行状态
- 可选增强能力仍依赖第三方组件，详见 `THIRD_PARTY_NOTICES.md`
